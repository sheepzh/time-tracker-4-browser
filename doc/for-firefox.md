## Building for Firefox

### Prerequisites

+ Node.js v22

### Build Steps

```shell
npm install
npm run build:firefox
```

### Output Files

+ Build directory: `dist_prod_firefox`

+ Marketplace package: `market_packages/target.firefox.zip`
